# 💻 Agência de Social Media - www.benattisocialmedia.com.br

Projeto desenvolvido com Next.js para apresentar uma agência de Social Media moderna, focada em posicionamento digital, gestão de redes sociais e crescimento de marcas.

## Descrição do Projeto
Este projeto é um site para uma Agência de Social Media, desenvolvido para apresentar serviços digitais, fortalecer o posicionamento online da marca e gerar conversão de clientes através de uma interface moderna, responsiva e profissional.

O projeto foi desenvolvido utilizando Next.js, React.js, TypeScript, Tailwind CSS, Node.js e outras tecnologias modernas voltadas para performance, responsividade e otimização SEO.

O projeto foi criado com o objetivo de ajudar empresas e marcas a fortalecerem sua presença digital, transmitindo autoridade, profissionalismo e aumentando o alcance nas redes sociais através de uma plataforma moderna e estratégica.
